# Daily Reflection & Self-Improvement Decision Tree

## Overview
This project is a deterministic decision tree designed to help users reflect on their daily activities and improve productivity and mental well-being.

## Features
- Structured decision tree logic
- Input validation (guardrails)
- Actionable outputs
- Python implementation

## How to Run
1. Download the file
2. Run:
   python decision_tree.py

## Output
The system asks questions and provides suggestions based on user input.
